# 🚀 AlphaDesk

Sistema de gerenciamento de chamados com autenticação de usuários.

## 🧰 Tecnologias

* Node.js
* Express
* SQLite
* JWT
* HTML, CSS e JavaScript

## 🔐 Funcionalidades

* Cadastro de usuários
* Login com autenticação JWT
* CRUD de chamados
* Multiusuário (cada usuário vê apenas seus chamados)

## ▶️ Como rodar o projeto

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

Abra o arquivo `index.html`
